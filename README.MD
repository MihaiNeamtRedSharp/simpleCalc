## Getting Started

1. Install NPM packages
   ```sh
   npm install
   ```
2. Run the tests
   ```sh
   npm test
   ```
3. Start the server
   ```sh
   npm start
   ```

   
<!-- USAGE EXAMPLES -->
## Usage

1. Addition
   ```sh
   GET http://localhost:3001/add?array=20,2,5

   {
    "sum": 27
    }
   ```
2. Subtraction
   ```sh
    GET http://localhost:3001/sub?array=20,2,5

   {
    "diff": 13
    }
   ```
3. Multiplication
   ```sh
     GET http://localhost:3001/mul?array=20,2,5

   {
    "prod": 200
    }
   ```
4. Division
   ```sh
     GET http://localhost:3001/div?array=20,2,5

   {
    "quot": 2
    }
   ```
